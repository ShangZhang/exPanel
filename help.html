Overview of exPanel

evaluate exRNA biomarker for liquid biopsy (exPanel) is a comprehensive open resource for exploring non-invasive biomarkers for 31 types of human diseases, including cancers, precancerous lesions, cardiovascular diseases, and so on. Currently, exPanel contains a large number of biomarkers for liquid biopsy. We manually curate 72 FDA-approved biomarkers form FDA official website, 1232 DNA, RNA, protein, and metabolite preclinical biomarkers from 745 papers. Additionally, a large patient cohort of 7 specimen types, 4023 RNA-seq samples were collected in exPanel, and approximately 45,000 RNA biomarker candidates were identified from these public RNA-seq data in the liquid biopsy field. All these markers are organized by four evidence levels to help users quickly evaluate the potential of a specific biomarker across different disease types.

exPanel provides a user-friendly and intuitive web interface and three function modules including Search, Browse, and Source to aid users in investigating liquid biopsy-based biomarkers. Browse module, which allows users to quickly browse biomarkers across different molecular types, disease types, and evidence levels. Search module, which returns a wealth of information for the queried biomarker, including basic information, evidence level, expression information (expression abundance and differential expression), diagnostic performance, biomarker reports, and FDA approval information. Expression information and diagnostic performance are only provided for RNA markers. Source module, which allows users to get to know the key characteristics of each dataset collected in exPanel. Users can also download RNA expression matrices and differential expression analysis results for the datasets of interest.

We anticipate that exPanel will serve as a valuable and powerful platform for developing non-invasive biomarkers across various human diseases.

Search module

Several options, including ‘RNA markers’, ‘DNA markers’, ‘Protein markers’, ‘Metabolite markers’ are available in the ‘Search’ module. Here, we illustrate an example to show how to search for RNA markers in exPanel, other types of markers can be searched in a similar way.

1.Input

Step 1:

Click the ‘RNA markers’ tab in the top button group. Four tabs represent different types of biomarkers, and users can click one type of interest.

Step 2:

Select an RNA type you are interested in, for example, “mRNA”. Once you choose the RNA type, the prompt box above it will show the corresponding search tips. Please notice that the default setting is ‘miRNA’.

Step 3:

Type the marker name of your interest according to the tips in the prompt box. For example, you can type ‘KRT19’. Available items will be automatically displayed to assist in your input.

Step 4:

Click the ‘Search an RNA biomarker’ button, then search results will be displayed.

2.Results:

2.1 Basic information.

Summary of KRT19 gene information, including the gene name, gene id, gene type, and gene length. We provide several external links (e.g. NCBI, Ensembl, GeneCards, etc.) for detailed annotation about the queried gene.

2.2 Evidence level.

Here, exPanel provides an interactive table to show the evidence level of the queried gene in different diseases which are available in our database. Evidence level is very important for users to evaluate the biomarker potential.

2.3 Expression information.

This section provides expression abundance and differential expression information on the queried gene. It includes:

1) Disease: disease type

2) Datasets: dataset accession id or data source name

3) RNA source: specimen type (e.g. Plasma Cell-free)

4) Comparison: negative class vs. positive class (e.g. Normal vs. Colorectal cancer)

5) Sample size: the number of clinical samples (negative class vs. positive class)

6) Detection frequency: the percentage of samples with normalized expression value (RPM/TPM) > 0

7) Expression N: the mean of each samples’ normalized expression value (RPM/TPM) of the negative class

8) Expression P: the mean of each samples’ normalized expression value (RPM/TPM) of the positive class

9) Bar chart: click this button, then a bar chart showing the normalized expression value (RPM/TPM) of each sample will be displayed.

10) logFC：the log2(fold change) of positive class vs. negative class

11) P-Value

12) FDR: false discovery rate

13) Box plot: click this button, then a box plot showing the expression level (RPM/TPM) of samples for the two classes (negative class vs. positive class) will be displayed.

2.4 Diagnostic performance.

This section provides diagnostic performance information to help users get to know the potential of the specific biomarker.

1) Disease: disease type

2) Datasets: dataset accession id or data source name

3) RNA source: specimen type (e.g. Plasma Cell-free)

4) Comparison: negative class vs. positive class (e.g. Normal vs. Colorectal cancer)

5) Sample size: the number of clinical samples (negative class vs. positive class)

6) AUC: area under the ROC curve

7) Precision: positive predictive value

8) Specificity: true negative rate

9) Sensitivity: true positive rate

10) ROC curve: receiver operating characteristic curve

2.5 Biomarker reports.

This section lists core information manually curated from literatures related to this specific biomarker across various diseases.

1) Reference: the research article and the link to PubMed

2) HGNC_Symbol: the approved symbol of the biomarker provided by HGNC

3) Biomarker_Name: the biomarker name reported by the original research article

4) Specimen: specimen type

5) Expression_Trend: expression change trend of the biomarker for positive class vs. negative class (cancer vs. normal, etc.)

6) Clinical_Value: potential clinical value of the biomarker(diagnosis/prognosis)

7) Discrimination/Prediction: the cohorts that can be distinguished by the diagnostic biomarker or the endpoint of the prognostic biomarker

8) Early_Stage: whether the biomarker can be used for early diagnosis (Y for yes, N for no, and Not involved for prognostic biomarker or not available.)

9) Etiology: pathogen of the specific disease, such as HBV-related HCC

10) Cellular_Effect: the cellular function of the biomarker

11) Sample_Size: the number of clinical samples of each cohort

12) Method: detection methods for the biomarker

13) Single/Panel: the type of biomarker model, single or panel

14) Journal: journal that published the biomarker research

15) Publish_Year: particular year of the biomarker research published

16) Region: country or population of the patient cohort

17) Disease_Type: disease type that the biomarker can be used for diagnosis or prognosis

18) Usage: the author’s summary description of the biomarker

2.6 FDA approval.

Here, we display FDA approval information manually curated from FDA official website related to this specific biomarker across various diseases.

1) HGNC_Symbol: the approved symbol of the biomarker provided by HGNC.

2) Official_Full_Name: the official full name of the biomarker provided by HGNC.

3) Marker_Name: common used name of the biomarker

4) Marker_Type: molecular type of the biomarker

5) Methodology: detection methods for the biomarker

6) Specimen: specimen type

7) Cancer_Type: cancer type that the biomarker can be used for diagnosis or prognosis.

8) Biomarker_usage: clinical usage of the biomarker

9) Submission_ID: submission id for FDA

10) Class: the classification of the medical device by FDA

11) Year_First_Approved: particular year of the biomarker first approved or cleared by FDA

12) Evidence_Level: one of the four evidence levels is assigned to the biomarker based on our classification system in exPanel.

Browse module

The data source is a matter of great account for our database. To this end, we provide an interactive table to summarize the sample information of the high-throughput sequencing data, and two bar charts to show detailed information. Key characteristics of each dataset are listed in an interactive table to help users get to know the data collected in exPanel. Users can click ‘Accession_ID’ to view the raw data page in GEO, SRA, BioProject, etc. Furthermore, users can also click ‘PMID’ to view the original article in PubMed. exPanel also provides expression matrices and differential expression analysis results from standard and consistent RNA-seq data processing pipelines. Read counts for each gene were normalized to RPM values (Reads Per Million) for small RNA and TPM values (Transcript Per Million) for long RNA. Differential expression results were categorized by available comparisons, such as normal versus cancer, precancerous lesions versus cancer, and so on. The expression matrices and differential expression analysis results of your interest can be easily downloaded by click the ‘Download’ button.

Source module

exPanel contains RNA, DNA, protein, and metabolite biomarkers across various diseases. And the corresponding tabs are available in the ‘Browse’ module. Here, we take RNA as an example to show how to browse biomarkers in exPanel, other types of makers can be browsed in a similar way.

1.Filter

Step 1:

Click the ‘RNA markers’ tab in the top button group. Four tabs represent different types of biomarkers, and users can click one type of interest.

Step 2:

Select an RNA type of your interest in the corresponding filter box (e.g. mRNA).

Step 3:

Or select a disease type of your interest in the corresponding filter box (e.g. Liver cancer).

Step 4:

Or select an evidence level of your interest in the corresponding filter box (e.g. Level 2).

2.Results:

Users can browse biomarkers of interest by biomarker name, RNA type, disease type, and evidence level. Evidence level is assigned to each biomarker in exPanel to help users quickly evaluate the biomarker potential across different disease types. By clicking the biomarker name, search module will be displayed for searching detailed information about this marker.

Input description

For RNA:

Search mRNA, lncRNA, snRNA, snoRNA, and srpRNA by entering a gene symbol [e.g. B2M, HOTAIR, RN7SK, SNORA55, RN7SL1].

Search miRNA by entering a miRBase ID [e.g. hsa-miR-21-5p, hsa-miR-4482-3p，hsa-miR-4473].

Search piRNA by entering a piRNABank accession number [e.g. hsa_piR_000794, hsa_piR_000823, hsa_piR_002286].

Search circRNA by enitering a circBase identifier [e.g. hsa_circ_0000119, hsa_circ_0051802].


For DNA:

Search a DNA biomarker by entering a gene symbol [e.g. BRCA1, TP53, KRAS, etc.].


For protein:

Search a protein biomarker by entering a protein name [e.g. CA19-9, PSA, CEACAM5, etc.].


For metabolite:

Search a metabolite biomarker by entering a metabolite name [e.g. alanine, valine, tryptophan, etc.].
